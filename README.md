## 🚀 Overview
This is a **Netflix Clone** built using HTML, CSS, and JavaScript.  
It replicates the modern Netflix UI with responsive design and dark mode functionality.

## ✨ Features
- 🎥 Netflix-style UI design
- 🌙 Dark Mode Toggle
- 📱 Fully Responsive (Mobile + Desktop)
- 🎬 Video inside TV animation
- ⚡ Fast and lightweight
